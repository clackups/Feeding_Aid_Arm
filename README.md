# Feeding_Aid_Arm

This is a simple mechanical device that aims to facilitate the eating for a person with limited arm funcion, such as those suffering from cerebral palsy.

It is made from photography accessories as listed below and is easy to assemble.

## Part list

### C-Clamp with 1/4" 3/8" Thread Hole

A clamp with a 3/8" mound hole and enough travel to fixate a fork or a spoon. [Amazon](https://www.amazon.com/C-Clamp-C-Shaped-Desktop-Monitor-Fixture/dp/B0FT5ZPKXR), [AliExpress](https://www.aliexpress.us/item/3256808835661051.html)

There are many varieties of such a clamp on the market. Importan is, that the shaft can go all the way to the lower edge, so that it can fixate your utensil firmly enough.
